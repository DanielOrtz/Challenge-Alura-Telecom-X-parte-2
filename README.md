# Proyecto Telecom-X - Análisis de Evasión de Clientes (Churn)
📊 Predicción de Cancelación de Clientes – Telecom X
📌 Descripción del proyecto

Este proyecto forma parte del Challenge de Data Science de Alura LATAM y Oracle Next Education.

El objetivo es desarrollar modelos de Machine Learning capaces de predecir qué clientes tienen mayor probabilidad de cancelar su servicio (churn) en la empresa ficticia Telecom X.

La predicción del churn es clave para las empresas de telecomunicaciones, ya que permite identificar clientes en riesgo y aplicar estrategias de retención antes de que abandonen el servicio.

Este proyecto incluye el preprocesamiento de datos, entrenamiento de modelos de clasificación, evaluación de métricas y análisis de variables importantes para comprender los factores que influyen en la cancelación.

🎯 Objetivos

Preparar los datos para modelos de Machine Learning

Analizar la correlación entre variables

Entrenar modelos de clasificación

Evaluar el desempeño mediante métricas

Identificar los factores que influyen en la cancelación de clientes

Generar insights estratégicos para la retención

Estas herramientas permiten realizar análisis de datos, visualización y construcción de modelos predictivos dentro del flujo de trabajo de ciencia de datos.

⚙️ Proceso del proyecto

El análisis se realizó siguiendo un flujo típico de proyectos de Data Science:

1️⃣ Carga y exploración de datos

Se importaron los datos y se realizó una exploración inicial para comprender su estructura, tipos de variables y posibles problemas en el dataset.

2️⃣ Preprocesamiento de datos

Se aplicaron técnicas de preparación de datos:

Limpieza de datos

Codificación de variables categóricas

Normalización de variables

División del dataset en entrenamiento y prueba

3️⃣ Análisis de correlación

Se analizaron las relaciones entre variables para identificar aquellas que tienen mayor impacto en la cancelación de clientes.

4️⃣ Entrenamiento de modelos

Se entrenaron diferentes modelos de clasificación para predecir el churn.

Ejemplos de modelos utilizados:

Regresión Logística

Random Forest

5️⃣ Evaluación de modelos

Los modelos se evaluaron mediante métricas como:

Accuracy

Precision

Recall

F1 Score

Matriz de confusión

Esto permitió comparar el desempeño de cada modelo y seleccionar el más adecuado.

6️⃣ Importancia de variables

Se analizó la importancia de las variables para entender qué factores influyen más en la cancelación de clientes.

Algunos factores relevantes identificados incluyen:

Tipo de contrato

Cargos mensuales

Tiempo de permanencia del cliente

Servicios adicionales contratados

Estos factores ayudan a entender el comportamiento de los clientes y permiten diseñar estrategias de retención.

📊 Resultados

El modelo desarrollado permite identificar clientes con alta probabilidad de cancelar el servicio, lo que permite a la empresa tomar decisiones estratégicas basadas en datos.

Entre los hallazgos principales:

Los clientes con contratos mensuales tienen mayor probabilidad de cancelar.

Los clientes con poca antigüedad en la empresa presentan mayor riesgo de churn.

Algunos servicios adicionales influyen en la fidelización del cliente.

💡 Conclusión

La aplicación de modelos de Machine Learning permite anticipar el comportamiento de los clientes y detectar patrones de cancelación.

Con esta información, la empresa puede:

Implementar estrategias de retención temprana

Ofrecer promociones personalizadas

Mejorar la experiencia del cliente

Este tipo de soluciones basadas en datos ayudan a reducir la pérdida de clientes y mejorar la rentabilidad del negocio.
